# 📊 Торговый журнал — Telegram Bot

## Быстрый запуск локально

1. Установи зависимости:
```
pip install -r requirements.txt
```

2. Запусти:
```
python bot.py
```

## Деплой на Railway

1. Загрузи все файлы на GitHub (новый репозиторий)
2. Зайди на railway.app → New Project → Deploy from GitHub
3. Выбери репозиторий
4. В разделе Variables добавь:
   - BOT_TOKEN = твой токен от BotFather
5. Railway сам запустит бота

## Команды бота

- /start — главное меню
- /cancel — отменить добавление сделки
- /delete_ID — удалить сделку (например /delete_5)

## Функции

- ➕ Добавить сделку (пара, направление, вход/выход, лот, результат, комментарий)
- 📊 Статистика (винрейт, P&L, лучшая/худшая пара)
- 📅 Календарь (навигация по месяцам, 🟢 отмечает дни со сделками)
- 📋 История последних 20 сделок
